### s3.py

``` bash
usage: main.py [-h] [--keep_bucket] bucket_name region

positional arguments:
  bucket_name    The name of the bucket to create.
  region         The region in which to create your bucket.

optional arguments:
  -h, --help     show this help message and exit
  --keep_bucket  Keeps the created bucket. When not specified, the bucket is
                 deleted at the end of the demo.
```
